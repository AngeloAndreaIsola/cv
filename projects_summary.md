# Projects Summary

### CAT Suite
* **Elevator Pitch:** Enterprise procurement AI platform for automated transaction classification, grouping, and analytics.
* **Tech Stack:** Python, Django, FastAPI, BERT, FAISS, GCP (Vertex AI, Cloud Run), MySQL, PostgreSQL, Cypress.
* **Key Achievements:**
    * Built ML-driven procurement classification pipelines.
    * Developed automated parsing and ingestion for varied enterprise datasets.
    * Implemented enterprise-wide deployment automation (Terraform/Cloud Build).
    * Created interactive dashboarding for taxonomy management and review.

### GroceryApp / Recipe Platform
* **Elevator Pitch:** Smart meal planning and grocery platform with automated source discovery.
* **Tech Stack:** Python (FastAPI), React, TypeScript, Vite, PostgreSQL, Redis, Docker.
* **Key Achievements:**
    * Designed full-stack architecture for recipe meal planning.
    * Implemented intelligent scraping for grocery list enhancement.
    * Optimized UI for performance with React and Zustand.

### Wishlist App
* **Elevator Pitch:** Self-hosted personal wishlist manager with automated product scraping.
* **Tech Stack:** Python (FastAPI), React, Playwright, PostgreSQL.
* **Key Achievements:**
    * Developed automated product metadata extraction engine.
    * Architected simple, portable self-hosted deployment pattern.

### WhichLLM
* **Elevator Pitch:** CLI utility for ranking local LLMs based on user hardware capabilities.
* **Tech Stack:** Python, Hugging Face, GGUF.
* **Key Achievements:**
    * Developed automated resource detection (RAM/GPU).
    * Implemented hardware-aware ranking algorithms.
